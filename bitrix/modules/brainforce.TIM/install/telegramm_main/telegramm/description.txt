BrainBot.php - отвечает за приём запросов от бота.
index.php - Страница для создания кода валидации и редиректа на страницу подключения бота.
telegram-connect.php - Страница с кнопкой "Подключить".
config.php - Файл с константами.

Папка Bot:
BotMassage.php - Генерирует сообщения и отправляет их в чат с пользователем. 
Functions.php - Генерирует проверяет и записывает код валидации. Берёт параметры передаваймые коммандой /start.
Validations.php - Записывает код чата.
